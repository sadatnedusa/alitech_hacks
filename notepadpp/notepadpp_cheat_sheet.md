## **Notepad++ cheat sheet** to help you use the editor more efficiently:
---

### **1. Basic Text Editing**
| **Action**                       | **Shortcut**                |
|----------------------------------|-----------------------------|
| **New File**                     | `Ctrl + N`                  |
| **Open File**                    | `Ctrl + O`                  |
| **Save File**                    | `Ctrl + S`                  |
| **Save As**                      | `Ctrl + Alt + S`            |
| **Close File**                   | `Ctrl + W`                  |
| **Close All Files**              | `Ctrl + Shift + W`          |
| **Undo**                         | `Ctrl + Z`                  |
| **Redo**                         | `Ctrl + Y`                  |
| **Cut**                          | `Ctrl + X`                  |
| **Copy**                         | `Ctrl + C`                  |
| **Paste**                        | `Ctrl + V`                  |
| **Select All**                   | `Ctrl + A`                  |
| **Delete Line**                  | `Ctrl + L`                  |
| **Duplicate Line**               | `Ctrl + D`                  |
| **Move Line Up/Down**            | `Ctrl + Shift + Up/Down`    |
| **Comment/Uncomment Line**       | `Ctrl + Q`                  |
